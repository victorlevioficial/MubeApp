# Review de Icones da Busca

Esta pasta serve para revisar os SVGs antes de levar qualquer asset para o app.

## Como ver

1. Abra [index.html](/mnt/c/Users/Victor/Desktop/AppMube/design_review/search_filter_icons/index.html) no navegador.
2. Use o link `Abrir SVG` em qualquer card para abrir o arquivo cru.
3. Se quiser regenerar os assets, rode:

```bash
./scripts/build_search_filter_icon_review.sh
```

## O que tem aqui

- `index.html`: galeria local para review visual
- `icons/`: SVGs candidatos por filtro
- `SOURCES.md`: origem de cada arquivo
- `LICENSE.FontAwesome.txt`: licenca trazida do repositorio oficial usado no review

## Escopo

Os arquivos desta pasta sao para validacao visual.
Eles ainda nao estao conectados ao Flutter.
