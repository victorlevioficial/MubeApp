Review local de SVGs para os filtros da busca

Esta galeria existe para aprovar os icones antes de levar qualquer asset para o Flutter. Cada card mostra uma previa no clima visual do app, uma visualizacao do SVG cru e um link para abrir o arquivo direto.

Sem abrir o app SVGs locais em design_review/ Base oficial: Font Awesome Free 7.x
Abra qualquer link Abrir SVG para inspecionar o arquivo individualmente. A origem detalhada de cada asset esta em SOURCES.md.
