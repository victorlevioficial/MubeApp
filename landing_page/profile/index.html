Perfil compartilhado do Mube

Abra este perfil direto no app. Se o app não estiver instalado, continue pelo site e baixe o Mube.

Os links de perfil do Mube funcionam como App Link e Universal Link. Em celulares compatíveis, tocar no link abre o perfil no app. Se você caiu no navegador, use os atalhos abaixo para tentar novamente.

Abrir no app Baixar no Google Play Ir para o site do Mube
Android App Links com verificação de domínio para `mubeapp.com.br`.
iPhone Universal Links configurados pelo domínio associado.
Fallback web O perfil compartilhado continua acessível mesmo fora do app.

Em alguns cenários, o sistema operacional precisa terminar a verificação do domínio ou o app precisa já ter sido aberto ao menos uma vez após a instalação.
